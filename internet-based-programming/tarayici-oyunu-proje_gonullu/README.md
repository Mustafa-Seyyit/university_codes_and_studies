Kod Avi is a simple browser-based reflex game built with HTML, CSS, and JavaScript.
The player clicks the moving code symbol and tries to get the highest score within 30 seconds.

### Features

- 4-page structure: Home, Game, Rules, Contact
- Simple and clear game logic
- Responsive (mobile-friendly) layout
- Contact page with a form

### Run

1. Open the project folder.
2. Launch `index.html` in a browser.
3. Go to the "Oyunu Oyna" page to start playing.

# Kod Avi - Browser Game Project

🇹🇷
Kod Avi, HTML, CSS ve JavaScript ile gelistirilmis basit bir tarayici refleks oyunudur.
Oyuncu, oyun alaninda rastgele beliren kod sembolune tiklayarak 30 saniye icinde en yuksek puani yapmaya calisir.

### Ozellikler

- 4 sayfali yapi: Ana Sayfa, Oyun, Kurallar, Iletisim
- Basit ve anlasilir oyun mekanigi
- Responsive (mobil uyumlu) tasarim
- Form iceren iletisim sayfasi

### Calistirma

1. Proje klasorunu acin.
2. `index.html` dosyasini tarayicida baslatin.
3. Oyuna gecmek icin "Oyunu Oyna" sayfasini acin.
